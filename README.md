# Midori
Brand spanking new Nano with 100% less shit code

### Installation
**macOS**
```bash
brew install pkg-config cairo libpng jpeg giflib ffmpeg
npm install
```

**Ubuntu**
```bash
sudo apt install libcairo2-dev libjpeg8-dev libpango1.0-dev libgif-dev build-essential ffmpeg g++
npm install
```
